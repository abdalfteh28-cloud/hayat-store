# تسجيل الدخول وربط المشروع بـ Vercel

اتبع الخطوات بالترتيب. الربط يتم مرة واحدة فقط.

---

## الطريقة 1: من ملف التشغيل (الأسهل)

1. **شغّل الملف:** `تشغيل — تسجيل الدخول وربط Vercel.bat`  
   (دبل كليك من مستكشف الملفات)

2. **تسجيل الدخول:**
   - ستُفتح نافذة المتصفح أو يظهر **رابط** في الطرفية.
   - افتح الرابط في المتصفح وسجّل الدخول بحسابك في Vercel (أو أنشئ حساباً).
   - بعد تسجيل الدخول ارجع للطرفية واضغط Enter إذا طُلب منك.

3. **ربط المشروع:**
   - يطلب منك اختيار **الفريق (Team)** أو الحساب الشخصي.
   - يطلب اختيار **المشروع** أو إنشاء مشروع جديد — اختر "Link to existing project" أو "Create new project" حسب رغبتك.
   - أدخل الاسم إذا طُلب منك.

4. عند انتهاء التنفيذ بدون أخطاء = **تم الربط**. بعدها يمكنك الرفع بـ `/vercel-deploy` من Cursor أو `npm run vercel`.

---

## الطريقة 2: من الطرفية في Cursor

1. افتح الطرفية: **Terminal → New Terminal** (أو **Ctrl+`**).

2. تأكد أنك داخل مجلد المشروع، ثم نفّذ:

   ```bash
   npm install
   ```

3. تسجيل الدخول:

   ```bash
   npx vercel login
   ```
   - افتح الرابط الذي يظهر وسجّل الدخول في المتصفح، ثم ارجع للطرفية.

4. ربط المشروع:

   ```bash
   npx vercel link
   ```
   - اختر الفريق/المشروع أو أنشئ مشروعاً جديداً كما يطلب منك.

---

## بعد الربط

- للرفع التجريبي: `npm run vercel` أو من Cursor: **`/vercel-deploy`**
- للرفع كإنتاج: `npm run vercel:deploy`

إذا واجهت رسالة خطأ، تأكد أن **Node.js** مثبت من [nodejs.org](https://nodejs.org) وأن الطرفية تتعرف على `node` و `npm` (جرّب: `node -v` و `npm -v`).
