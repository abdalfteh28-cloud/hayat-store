# ربط المشروع بـ Vercel من Cursor

## من الـ Marketplace (إضافة Cursor)

إضافة **Vercel** التي ثبّتها من Cursor Marketplace لا تثبت الـ CLI ولا تربط المشروع تلقائياً.  
الربط يتم من خلال **تسجيل الدخول** و**ربط المشروع** مرة واحدة.

---

## ماذا تفعل الآن؟ — تسجيل الدخول والربط

1. **شغّل الملف:** `ربط Vercel — تسجيل الدخول.bat` (دبل كليك).
2. السكربت سيقوم بـ:
   - تثبيت الاعتماديات (بما فيها Vercel CLI) إن لزم.
   - **يطلب منك تسجيل الدخول:** ستُفتح المتصفح أو يظهر رابط — سجّل الدخول بحسابك في Vercel.
   - **ربط المشروع:** يطلب منك اختيار الفريق (Team) والمشروع أو إنشاء مشروع جديد.
3. بعد انتهاء السكربت يصبح المشروع **مرتبطاً** بحسابك، ويمكنك الرفع من Cursor (مثلاً أمر الـ deploy من إضافة Vercel) أو بالأوامر:
   - `npm run vercel` — رفع تجريبي
   - `npm run vercel:deploy` — رفع إنتاج

---

## ملخص

| الخطوة | ماذا يحدث |
|--------|-----------|
| تشغيل `ربط Vercel — تسجيل الدخول.bat` | يطلب منك **تسجيل الدخول** ثم **ربط** المشروع |
| بعد الربط | يمكنك الرفع من Cursor أو بـ `npm run vercel` |

إضافة الـ Marketplace في Cursor جاهزة؛ الربط الفعلي يتم مرة واحدة عبر السكربت أعلاه.
