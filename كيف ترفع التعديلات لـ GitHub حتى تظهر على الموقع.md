# لماذا لا تظهر التعديلات على الموقع؟

الموقع **https://hayat-store.onrender.com** يعرض الكود الموجود على **GitHub**، وليس الملفات من جهازك مباشرة.  
كل التعديلات التي تمت (شريط العروض، لماذا يثق العملاء، الأسئلة الشائعة، الداشبورد، إلخ) موجودة في مجلدك **«مجلد جديد»** لكنها **لم تُرفع بعد** إلى المستودع على GitHub.

---

## الطريقة ١ — تثبيت Git ثم الرفع من الطرفية

### 1) تثبيت Git
- حمّل من: **https://git-scm.com/download/win**
- ثبّت البرنامج (Next حتى النهاية).
- أعد تشغيل الطرفية (PowerShell أو CMD) إن لزم.

### 2) من مجلد المشروع نفّذ:

افتح **PowerShell** أو **CMD** ثم:

```powershell
cd "c:\Users\hardw\OneDrive\Documents\مجلد جديد"
git add .
git status
git commit -m "تحديث: شريط عروض، لماذا يثق العملاء، أسئلة شائعة، داشبورد"
git push origin main
```

- إذا طلب منك **اسم المستخدم وكلمة المرور** فاستخدم حساب GitHub (أو Personal Access Token بدل كلمة المرور).
- إذا ظهرت رسالة أن الفرع اسمه `master` بدل `main` فاستخدم: `git push origin master`

### 3) بعد الرفع
- ادخل إلى **Render** → خدمة hayat-store: غالباً سيبدأ **نشر تلقائي** (Deploy).
- انتظر 2–5 دقائق ثم افتح: **https://hayat-store.onrender.com** وحدّث الصفحة (F5 أو Ctrl+F5).

---

## الطريقة ٢ — رفع الملفات يدوياً من موقع GitHub

إذا لم ترد تثبيت Git:

1. ادخل إلى **https://github.com/abdalfteh28-cloud/hayat-store**
2. افتح الملفات التي تغيّرت (مثل **index.html**، **styles.css**، **admin.html**).
3. اضغط على أيقونة **القلم (Edit)**.
4. **احذف** المحتوى القديم والصق المحتوى من الملف نفسه من مجلدك **«مجلد جديد»** (افتح الملف بنوتبد أو Cursor وانسخ الكل).
5. اضغط **Commit changes**.
6. كرّر لكل ملف تم تعديله.

الملفات التي تم تعديلها أو إضافتها:
- **index.html**
- **styles.css**
- **server.js**
- **admin.html** (ملف جديد)

بعد رفعها كلها، Render سيعيد النشر تلقائياً (إن كان مربوطاً بالمستودع) وستظهر التعديلات على الموقع.

---

## ملخص

| أين التعديلات؟ | على جهازك في «مجلد جديد» |
| أين يقرأ الموقع؟ | من GitHub (المستودع abdalfteh28-cloud/hayat-store) |
| ماذا تفعل؟ | ارفع ملفات «مجلد جديد» إلى GitHub (بالطريقة ١ أو ٢) |

بعد الرفع، الرابط يبقى: **https://hayat-store.onrender.com**
