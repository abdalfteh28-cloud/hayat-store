# نشر المتجر على السيرفر — والرابط بعد النشر

لا يمكن النشر تلقائياً من هنا؛ النشر يتم **من عندك** عبر Git و Render. بعد أن تنفّذ الخطوات، الرابط يكون كالتالي.

---

## رابط المتجر (بعد النشر)

| الصفحة | الرابط |
|--------|--------|
| **الصفحة الرئيسية / المتجر** | **https://hayat-store.onrender.com** |
| **لوحة التحكم** | **https://hayat-store.onrender.com/admin** |

*(إذا غيّرت اسم الخدمة في Render، استبدل `hayat-store` باسم خدمتك.)*

---

## خطوات النشر (أنت تنفّذها)

### 1) رفع التعديلات إلى GitHub

من **PowerShell** أو **CMD**:

```bash
cd "c:\Users\hardw\OneDrive\Documents\مجلد جديد\متجر حياة"
git add .
git commit -m "تحديث المتجر بعد التعديلات"
git push origin main
```

إذا كان الفرع عندك `master` بدل `main`:

```bash
git push origin master
```

عند الطلب، أدخل اسم مستخدم GitHub وكلمة المرور (أو Token).

### 2) Render ينشر تلقائياً

- إذا كان المشروع مربوطاً بـ Render، بعد الـ `git push` يبدأ النشر تلقائياً.
- انتظر **2–5 دقائق** ثم افتح الرابط في المتصفح.

### 3) فتح الرابط

- المتجر: **https://hayat-store.onrender.com**
- لوحة التحكم: **https://hayat-store.onrender.com/admin**
