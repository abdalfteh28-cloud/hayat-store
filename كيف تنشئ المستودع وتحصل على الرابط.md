# كيف تنشئ المستودع (GitHub) وتحصل على عنوان الرابط

## لا يوجد مستودع جاهز الآن

المشروع موجود فقط على جهازك. **عنوان المستودع** (رابط Git) يظهر **بعد** أن تنشئ مستودعاً على GitHub وترفع المشروع.

---

## الخطوة ١ — إنشاء المستودع على GitHub

1. ادخل إلى **https://github.com** وسجّل الدخول (أو أنشئ حساباً).
2. اضغط الزر الأخضر **"New"** أو **"New repository"**.
3. املأ:
   - **Repository name:** مثلاً `hayat-store` (بدون مسافات)
   - **Description:** اختياري، مثلاً: متجر حياة — شرائح إنترنت
   - **Public**
   - **لا** تختر "Add a README" (المشروع عندك جاهز)
4. اضغط **"Create repository"**.

بعد الإنشاء تظهر لك صفحة فيها **عنوان المستودع**، مثل:

```
https://github.com/اسم-المستخدم-الذي-سجّلت-به/hayat-store
```

هذا هو **رابط المستودع (عنوان Git)** — انسخه لاستخدامه في Render أو لمشاركته.

---

## الخطوة ٢ — رفع المشروع من جهازك إلى المستودع

على جهازك، من **مجلد المشروع** (مجلد جديد الذي فيه `server.js` و `index.html`):

### إذا كان Git مثبتاً عندك:

```bash
cd "c:\Users\hardw\OneDrive\Documents\مجلد جديد"
git init
git add .
git commit -m "متجر حياة - أول رفع"
git branch -M main
git remote add origin https://github.com/اسمك/hayat-store.git
git push -u origin main
```

**غيّر** `اسمك/hayat-store` إلى **اسم مستخدمك** و**اسم المستودع** الذي أنشأته.

### إذا لم يكن Git مثبتاً:

- ثبّت Git من: **https://git-scm.com/download/win**
- ثم نفّذ الأوامر أعلاه من **موجه الأوامر** أو **PowerShell** من داخل مجلد المشروع.

## شكل عنوان المستودع (الرابط)

| نوع الرابط | الشكل |
|------------|--------|
| رابط المستودع في المتصفح | `https://github.com/اسمك/hayat-store` |
| رابط Git للرفع (HTTPS) | `https://github.com/اسمك/hayat-store.git` |

**اسمك** = اسم المستخدم في GitHub  
**hayat-store** = اسم المستودع الذي اخترته (يمكن تغييره)

---

## بعد الرفع

- **عنوان المستودع** الذي ترسله أو تستخدمه =  
  `https://github.com/اسمك/اسم-المستودع`
- في **Render**: تربط هذا المستودع (تختار من قائمة مستودعات GitHub) ولا تحتاج لكتابة الرابط يدوياً غالباً — فقط اختر المستودع من القائمة.
