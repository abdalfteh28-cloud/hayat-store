# تأكيد ربط المتجر بالباك إند وتشغيله

## 1. الربط بين الواجهة والباك إند

الواجهة (الفرونت إند) مربوطة بالباك إند عبر الطلبات التالية:

| الطلب        | الاستخدام |
|-------------|-----------|
| `GET /api/health` | التحقق من أن السيرفر يعمل |
| `GET /api/config` | جلب إعدادات الدفع و Google |
| `GET /api/shipping/options` | جلب خيارات الشحن عند فتح نموذج الطلب |
| `POST /api/order` | إرسال الطلب بعد تعبئة النموذج |
| `POST /api/create-payment` | إنشاء رابط الدفع (عند تفعيل Moyasar) |

كل هذه الطلبات تُرسل إلى **نفس عنوان الموقع** (مثلاً `http://localhost:3000`)، لذلك يجب فتح الموقع من خلال السيرفر وليس بفتح ملف `index.html` مباشرة.

---

## 2. تشغيل المتجر

### إذا كان Node.js مثبتاً

1. افتح **Command Prompt** أو **PowerShell** داخل مجلد المشروع.
2. نفّذ:
   ```bash
   npm install
   npm start
   ```
3. افتح المتصفح على: **http://localhost:3000**

أو استخدم الملف **تشغيل المتجر.bat** (دبل كليك) وسيقوم بالتثبيت والتشغيل إن وُجدت `node` و `npm` في المسار.

### إذا لم يكن Node.js مثبتاً

1. حمّل Node.js من: https://nodejs.org (النسخة LTS).
2. ثبّته ثم أعد فتح الطرفية (أو الجهاز إن لزم).
3. نفّذ من مجلد المشروع:
   ```bash
   npm install
   npm start
   ```
4. افتح المتصفح على: **http://localhost:3000**

---

## 3. كيف تتأكد أن الربط يعمل؟

- عند فتح الموقع من **http://localhost:3000** (بعد تشغيل `npm start`):
  - في أسفل الصفحة ستظهر عبارة **«متصل بالباك إند»** بلون أخضر إذا كان الاتصال ناجحاً.
- إذا فتحت الموقع بفتح ملف `index.html` مباشرة:
  - ستظهر رسالة أن الموقع **غير متصل**، ويُطلب منك تشغيل السيرفر وفتح الموقع من نفس العنوان.

---

## 4. اختبار سريع للباك إند

من المتصفح أو من أي برنامج يرسل طلبات HTTP:

- افتح: **http://localhost:3000/api/health**  
  إذا ظهر شيء مثل: `{"ok":true,"message":"الباك إند يعمل",...}` فالربط يعمل.

---

## 5. ملخص

- **الربط مؤكد** عندما تفتح الموقع من عنوان السيرفر (مثلاً `http://localhost:3000`) وتظهر «متصل بالباك إند» في الأسفل.
- **تشغيل المتجر** = تشغيل السيرفر بـ `npm start` ثم فتح **http://localhost:3000** في المتصفح.
